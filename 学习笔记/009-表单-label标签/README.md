## label标签

- 场景：常用于绑定内容与表单标签的关系  
- 标签名：label  

- 使用方法 1：
    > 1. 使用label标签把内容（如：文本）包裹起来  
    > 2. 在表单标签上添加id属性  
    > 3. 在label标签的for属性中设置对应的id属性值  

- 使用方法 2：  
    > 1. 直接使用label标签把内容（如：文本）和表单标签一起包裹起来  
    > 2. 需要把label标签的for属性删除即可
