## textarea文本域标签

- 场景：在网页中提供可输入多行文本的表单控件  
- 标签名：textarea  

- 常见属性：
    > - cols：规定了文本域内可见宽度  
    > - rows：规定了文本域内可见行数  

- 注意点：
    > - 右下角可以拖拽改变大小（可在css禁用）  
    > - 实际开发时针对于样式效果推荐用CSS设置
